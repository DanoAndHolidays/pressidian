项目简介 ​

Pressidian是一个使用Obsidian笔记仓库构建的VitePress静态站点，旨在提供一个简单、高效的知识管理和分享平台。

核心功能 ​

  • Obsidian集成：直接使用Obsidian笔记仓库，无需额外格式转换
  • 自动化部署：通过GitHub Actions实现自动构建和部署
  • 美观界面：基于VitePress构建，提供现代化的阅读体验
  • API支持：提供丰富的API接口，方便扩展和定制

快速开始 ​

  1. 克隆本仓库
  2. 安装依赖：npm install
  3. 启动开发服务器：npm run docs:dev
  4. 构建静态文件：npm run docs:build

技术栈 ​

  • VitePress
  • Vue.js
  • Markdown
  • GitHub Actions
